Oracle on-Prem to Azure DBAS: Technical Network Blueprint

Detailed Hub-Spoke Topology | L3 Routing Matrix | CIDR Allocation

Connectivity Stack

  • Azure ExpressRoute: L3 Private Peering, 10Gbps dedicated bandwidth.
  • Azure Hub VNet: Central routing hub with Azure Firewall and Gateway.
  • OCI FastConnect: L2 low-latency interconnect between Azure and OCI.

Target Environment

  • OCI Database@Azure: Managed service co-located in Azure regions.
  • Target DB: Oracle 19c Single Instance with Optimized Block Storage.
  • L3 Isolation: Dedicated CIDR 192.168.0.0/24 for target subnet.

Migration Lifecycle

  • Discovery: Automated scan of source DB environment and specs.
  • Staging: Provisioning target and preparing Physical Standby.
  • Cutover: Final synchronization and seamless role switchover.
